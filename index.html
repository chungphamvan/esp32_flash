Shino Web Flasher — ESP32-S3 (1-Click)

Tải file merged-binary.bin rồi nhấn Connect → Flash. Dùng Chrome hoặc Edge có hỗ trợ Web Serial API.

Nếu dùng file bootloader/partition/app riêng, hãy dùng ESP-IDF hoặc bật chế độ nâng cao.
Console ready. Use Chrome/Edge and allow access to USB device.
